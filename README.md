# MyOwn - 理性消费与价值感知

> 每一件物品都有其真实成本。MyOwn 帮助你从感性消费转向理性决策。

MyOwn 是一款轻量级的现代 Web 应用，旨在追踪物品的 **日均使用成本** 或 **次均使用成本**。通过极致的视觉反馈和工业级交互体验，让你清晰感知每一笔投入的实际价值。

## ✨ 核心特性

- **💰 真实成本计算**：基于购买天数或使用频率自动折算，让性价比清晰可见。
- **📱 移动端优先**：响应式毛玻璃（Glassmorphism）布局，针对手机端优化的抽屉交互。
- **🌈 AI 智能配色**：根据物品名称自动生成独一无二的和谐渐变。
- **⚡ 极致动效**：基于 Framer Motion 的顺滑过渡与触觉反馈（Active Scale）。
- **💾 零登录持久化**：使用 LocalStorage 存储，数据即开即用。

## 🛠️ 技术栈

- **框架**: Next.js 15 (App Router / Turbopack)
- **语言**: TypeScript 5
- **样式**: Tailwind CSS 4 (With CSS Variables)
- **动画**: Framer Motion & View Transition API
- **图标**: Lucide React
